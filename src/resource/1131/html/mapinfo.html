서울방면

고속버스 이용 시
강남터미널(서울⇒금산: 오전 6시 30분부터 오후 6시 30분까지 8회 운행) → 추부면에서 하차 (도보 10분 거리) → 본교
대전고속터미널 하차 → 대전 고속터미널 건너편 피자헛 앞 좌석버스 501번 이용 → 본교
기차 이용 시
서울역 → 대전역 하차 → 대전역 건너편 좌석버스 (501번) 이용 → 본교
승용차 이용 시
경부고속도로 이용 → 비룡분기점 (판암/무주방면) → 대진고속도로 진입 (무주방면) → 추부IC에서 하차 → 대전방면으로 3km 주행 (17번 국도) → 본 대학
