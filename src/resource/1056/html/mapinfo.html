대중교통을 이용하는 경우

교통편 오는방법
자동차 서울톨게이트(인터체인지) 방면
서울톨게이트 → 영동고속도로 → 북수원 IC → 의왕 IC → 과천의왕간 고속화도로 → 양재IC → 올림픽대로 → 동부간선도로 → 내부 순환도로 → 월곡역 → 동덕여자대학교

동서울터미널 방면
동서울터미널 → 강변북로 → 내부순환도로 → 월곡역 → 동덕여자대학교

김포공항 방면
김포공항 출발 → 양화대교ㅣ발산역 방면 → 가양대교 → 강변북로 → 내부 순환도로 → 월곡역 → 동덕여자대학교

강남고속터미널 방면
강남고속터미널 출발 → 반포로 → 올림픽대로 → 동부간선도로 → 내부 순환도로 → 월곡역 → 동덕여자대학교

서울역 방면
서울역 출발 → 퇴계로 → 청계천로 → 내부순환도로 → 월곡역 → 동덕여자대학교
지하철 6호선 월곡역(동덕여대역) 에서 하차 : 2번/3번 출구에서 500m 지점

동서울터미널 방면
강변역 → 신당역 환승 → 월곡역 하차 (32분 소요)

김포공항 방면
김포공항역 → 청구역 환승 → 월곡역 하차 (65분 소요)

강남고속터미널 방면
고속터미널역 → 약수역 환승 → 월곡역 하차 (30분 소요)

서울역 방면
서울역 → 동묘앞 환승 → 월곡역 하차 (25분 소요)
